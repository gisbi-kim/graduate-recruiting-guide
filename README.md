# Graduate Recruiting Guide

**Behavioral Interview to Research Recruiting Framework**

[Hosted guide](https://gisbi-kim.github.io/graduate-recruiting-guide/)

대학원생 선발을 위한 행동면접 기반 연구역량 평가 가이드라인입니다. 기업 채용에서 쓰이는 behavioral interview 관점을 연구실 맥락으로 재해석해, 후보자가 현재 완성된 연구자인지보다 **좋은 연구자로 성장할 가능성이 높은지**를 구조적으로 판단하는 데 초점을 둡니다.

작성자: **Giseop Kim**  
작성일: **2026-05-10**  
버전: **v0.1**

## Overview

이 문서는 대학원 연구실의 석사, 박사, 통합과정, 인턴 후보자를 평가할 때 사용할 수 있는 선발 기준과 면접 질문 세트를 제공합니다.

핵심 질문은 다음과 같습니다.

- 후보자가 연구 문제를 스스로 이해하고 정의할 수 있는가?
- 기술적 기본기와 학습 속도가 연구실의 기대 수준에 맞는가?
- 실패, 피드백, 협업, 윤리 문제를 연구자로서 다룰 수 있는가?
- 현재 실력보다 앞으로의 성장 가능성을 판단할 근거가 충분한가?
- 우리 연구실의 운영 방식과 지도 방식 안에서 잘 성장할 수 있는가?

## Contents

가이드는 다음 항목을 포함합니다.

- 선발 철학과 평가 관점
- 연구 동기, 기술 기본기, 문제 해결력, 기여도 설명 능력 평가
- 실패 경험, 피드백 수용성, 협업, 연구 윤리 기준
- 연구실 적합성 평가 프레임

- 5점 평가 루브릭

## Evaluation Framework

이 가이드는 후보자를 단일 인상으로 평가하지 않고, 여러 축으로 나누어 판단하도록 설계되어 있습니다.

| Axis | Focus |
| --- | --- |
| Research Motivation | 왜 연구를 하려는지, 문제의식이 있는지 |
| Technical Foundation | 기본 지식, 구현력, 학습 가능성 |
| Ambiguous Problem Handling | 정답이 없는 문제를 구조화하는 능력 |
| Contribution Clarity | 본인의 역할과 기여를 명확히 설명하는 능력 |
| Resilience | 실패와 막힘을 다루는 방식 |
| Coachability | 피드백을 받아 개선하는 태도 |
| Collaboration | 공동 연구와 갈등 상황에서의 행동 |
| Research Integrity | 신뢰성, 정직성, 재현성에 대한 태도 |

## Suggested Use

1. 면접 전에 후보자의 지원서, 포트폴리오, 프로젝트 경험을 확인합니다.
2. 문서의 질문 세트에서 후보자 유형에 맞는 질문을 고릅니다.
3. 면접 중 답변의 유창함보다 구체적 행동, 판단 근거, 반복되는 패턴을 기록합니다.
4. 면접 후 5점 루브릭과 위험 신호 기준으로 독립적으로 평가합니다.
5. 최종 의사결정 템플릿에 합격, 조건부 합격, 보류, 불합격 근거를 남깁니다.
